# 演示交付说明（一页纸）

局域网演示交接用。完整步骤见 [`第三方接入简要指南.md`](第三方接入简要指南.md)。公网差异见 [`第三方接入-公网部署要点.md`](第三方接入-公网部署要点.md)。

## 启动 Cloud

```powershell
cd fly-print-cloud
docker compose up --build -d
```

- 入口：`http://127.0.0.1:8012`（端口默认 **8012**，可用 `HTTP_PORT` 改）
- 管理端登录：用户名 **`admin`**，密码 **`admin123`**（演示默认，公网须改）
- 健康检查：`/health`、`/api/v1/health`
- 不必先复制 `.env`；需要改密码/端口时再 `Copy-Item .env.example .env`

## Demo

- 路径：`http://127.0.0.1:8012/integration-demo/`
- Setup（粘贴 HMAC）：`/integration-demo/setup`
- Provider code：`livacloud-demo`（管理端「三方接入」）

## Edge 安装包

- 版本：**1.0.38**
- 本地构建产物（不入库）：`fly-print-edge/dist/flyprint-edge-setup-1.0.38.exe`
- `cloud.base_url` 须为手机可达的局域网地址（不要给手机用 localhost）
