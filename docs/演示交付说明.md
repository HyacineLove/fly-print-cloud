# 演示交付说明（一页纸）

**主路径：** 局域网部署 Cloud + Edge，跑通官方扫码打印。  
**完整说明：** [`M0演示交接手册.md`](M0演示交接手册.md)。  
**第三方对接契约：** [`第三方接入指南.md`](第三方接入指南.md)（局域网 Demo 见其 §7）。

## 启动 Cloud

```powershell
cd fly-print-cloud
docker compose up --build -d
```

- 入口：`http://127.0.0.1:8012`（端口默认 **8012**，可用 `HTTP_PORT` 改）
- 管理端：用户名 **`admin`**，密码 **`admin123`**（演示默认，公网须改）
- 健康检查：`/health`、`/api/v1/health`
- 不必先复制 `.env`；Compose 会顺带起 Demo 容器，**主路径可忽略** `/integration-demo/`

## Edge

- 版本：**1.0.42**（`fly-print-edge/dist/flyprint-edge-setup-1.0.42.exe`，不入库；含占用 UX 收口）
- `cloud.base_url` = 手机可达的 Cloud 根（局域网 `http://IP:8012` 或公网 `https://域名`；不要给手机用 localhost；HTTPS 勿配 localhost 再改写）
- 激活 → 默认 IPP → 管理端可见（Edge↔Cloud 自动 `http↔ws` / `https↔wss`）
- 扫码进门后终端遮挡二维码；刷新作废手机会话。预览取消须确认；打印中不可中断回扫码。

## 必验收（M0-4b）

官方扫码打印至任务 **`completed`** 1 次。
